Latest Project

Stream-IT

Stream it is a web application made in Angular. It was my first big project that I wrote in Angular. The back-end of this web application is written in .NET 5.

You have the functionality to register, log in and sign out, the data will be stored in a SQL Database.

You can click on the poster of a movie, you will get a pop-up of the movie, which you can expand to full screen to fully enjoy the movie you have chosen. If you close the modal, the video will automatically stop playing and you can chose another movie to watch!

Technologies used:

  • HTML
  • CSS
  • Typescript
